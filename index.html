如何用Hugo搭建博客并部署到Github page

Author published on 2022-10-20
1. 安装Hugo Windows： 先安装choco包管理器，在管理员权限下运行cmd，执行如下命令： powershell -NoProfile -ExecutionPolicy unrestricted -Command "iex ((new-object net.webclient).DownloadString('https://chocolatey.org/install.ps1'))" # 设置环境变量 SET PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin 再使用choco 安装 hugo： choco install hugo -confirm MacOs： 使用brew 命令安装： brew install hugo Linux： 使用snap 命令安装： snap install hugo 检查是否安装成功： 输入: hugo version，若出现版本信息则表示安装成功 2. 新建站点 执行以下命令创建新站点： hugo new site myblog 这个命令会创建一个名为myblog的文件夹，这就是博客的根目录。站点目录结构如下： ├── archetypes │ └── default.md ├── config.toml # 博客站点配置文件 ├── content # 博客文章所在目录 ├── data ├── layouts # 网站布局 ├── static # 静态内容 └── themes # 博客主题 archetypes 内容模板文件，其中包含预配置的首选项（日期、标题、草稿等）。可以用自定义的预配置前端字段创建新的原型。
