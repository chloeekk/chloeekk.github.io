How to Use Skills in Claude Code?

Chloe published on 2026-04-25 included in AI

Claude Code is an agentic command-line tool introduced by Anthropic, and Skills are its core extension mechanism.

Simply put, Skills are packaged sets of instructions, scripts, and resources that teach Claude how to perform specific tasks in a standardized way.

What Are Claude Code Skills?

In Claude Code, Skills function like “professional plugins” or SOPs (Standard Operating Procedures) added to AI.

  • Essence: They are Markdown-based instructions stored in specific folders, sometimes including executable scripts.
  • Difference from Projects: “Claude Projects” typically refer to a feature in the Claude.ai web app, where you can upload documents, code files, and technical standards into a Project and define “Custom Instructions.” Projects usually provide static background knowledge (e.g., “this is our API documentation”), whereas Skills provide procedural knowledge (e.g., “generate a PR description following these 5 steps”).
  • Core value: Eliminates the need to repeatedly input the same preferences, workflows, or domain knowledge in every conversation.

CLI vs GUI vs Skills

Chloe published on 2026-03-29 included in AI

In 2026, mainstream AI productivity tools are undergoing a significant reconstruction of interaction logic. The release of Claude Code by Anthropic and Gemini CLI by Google marks a shift of core operating environments from traditional graphical user interfaces (GUI) back to command-line interfaces (CLI). The essence of this trend lies in improving the execution efficiency of complex tasks through the division of interaction layers.

Over the past decades, software development has primarily focused on lowering the barrier to entry through graphical interfaces. However, technological developments in 2026 show that leading AI tools are prioritizing the integration of command-line interfaces (CLI)—that is, controlling system interfaces through specific text-based instructions. This shift is not a regression, but rather a result of AI’s significantly higher speed and accuracy in processing text-based commands compared to simulating operations in visual interfaces.
